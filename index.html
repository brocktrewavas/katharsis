katharsis

augmented reality application designed for stress relief

Aim:

Katharsis is a speculative application which aims to provide an outlet to release built up stress from your daily life by giving you a consequence-free and fun way to break items. The application is based on cathartic therapy which states expressing aggression such as breaking objects reduces the feeling of anger and stress.

There’s no evidence to supporting cathartic therapy and it should not be used as a substitution to proper mental health care. Though people find expressing their aggression is helpful in the short term.

How to use:

Select an object from an array you’d like to destroy.
Gesture a throwing motion with your phone.
Watch the object fly through the projected virtual environment and break.

(Katharsis is not reliable for broken phones)

Technical:

Projector

Katharsis requires a 3D DLP data projector to project the virutal environment and holographic objects

3D Glasses

3D DLP projector glasses with the same refresh rate as the projector are required to connect to the data projector to view the environment and holographic objects

Mobile Phone

A mobile phone(with motion sensing compatiability) is required to connect to the projector and interact with the application.

Prototype:

Katharsis is a speculative based project. A functioning prototype is feasible but there is multiple technical challenges in terms of projecting and viewing the augmented reality environment and objects.

If the prototypes projection feature is unfeasible, the design will be adapted still utilize the phones motion sensors but to be viewed in a more simplified version of utilizing the mobiles camera and screen features to view a real environment without the need for a projector.

Katharsis will be developed in Unity 3D as a mobile phone application. If further developed beyond the prototype stage, gamification elements such as scoring, unlockable objects and different play styles will be added.

styling:

Katharsis’ design styling will have a simple and minimal aesthetic. The UI will contrast the complexity of setting up the application and allow for the user to easily access their tool of destruction. The colour scheme consists of blue colours to help psychologically aid the mind into becoming calm. The logo is designed to be a mix between the destruction of objects and the relief of stress.
